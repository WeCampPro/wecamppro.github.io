تصمیم سریع

این کمپ برای چه تغییری طراحی شده؟

کمپ مدیریت کلود وی‌کمپ یک برنامه آموزشی پروژه‌محور برای علاقه‌مندان و متخصصان فناوری اطلاعات است که می‌خواهند طراحی، پیاده‌سازی و نگه‌داری زیرساخت ابری IaaS را جدی‌تر یاد بگیرند.

مسیر دوره با مبانی لینوکس و شبکه شروع می‌شود و سپس به OpenStack، مفاهیم DevOps و ابزارهای عملی برای مدیریت زیرساخت‌های پیچیده می‌رسد.

این کمپ برای تبدیل دانسته‌های پراکنده زیرساختی به مهارت عملی مدیریت سرویس ابری طراحی شده است.

مناسب برای

  • افراد فعال در IT، شبکه یا سیستم که می‌خواهند وارد فضای Cloud شوند.
  • توسعه‌دهندگان یا ادمین‌هایی که به درک عملی زیرساخت ابری نیاز دارند.
  • تیم‌هایی که با OpenStack یا مفاهیم IaaS درگیر هستند.

پیش‌نیاز

  • آشنایی مقدماتی با لینوکس یا شبکه کمک‌کننده است.
  • انگیزه برای تمرین عملی و کار با خط فرمان لازم است.
