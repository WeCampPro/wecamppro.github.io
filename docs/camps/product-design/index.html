تصمیم سریع

این کمپ برای چه تغییری طراحی شده؟

کمپ دیزاین محصول وی‌کمپ برای یادگیری عملی طراحی محصول، تجربه کاربری، رابط کاربری و تفکر طراحی ساخته شده است. مسیر دوره از مفاهیم پایه شروع می‌شود و به پروژه نهایی، ارائه، پورتفولیو و آمادگی مصاحبه می‌رسد.

در نسخه قبلی کمپ، یادگیری به‌صورت هیبرید و همراه با کلاس‌های آنلاین، جلسات حضوری، ورکشاپ و منتورینگ پیش رفت. دوره آتی پس از نهایی شدن برنامه، با جزئیات جدید اعلام می‌شود.

هدف کمپ این است که یادگیری طراحی محصول به یک خروجی قابل ارائه برسد، نه فقط مجموعه‌ای از کلاس‌ها.

مناسب برای

  • افرادی که می‌خواهند از پایه وارد دیزاین محصول، UX یا UI شوند.
  • طراحانی که تجربه پراکنده دارند و به مسیر منسجم و پروژه‌محور نیاز دارند.
  • اعضای تیم‌های محصول که می‌خواهند زبان مشترک طراحی و کسب‌وکار را بهتر بفهمند.

پیش‌نیاز

  • آشنایی عمومی با محصول‌های دیجیتال
  • توانایی کار منظم روی تمرین‌ها و پروژه
  • دسترسی به لپ‌تاپ و ابزارهای طراحی
